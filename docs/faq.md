# Frequently Asked Questions

## General

### What is ProofPay?
ProofPay is a payment platform that uses on-chain attestations instead of traditional KYC processes.

### How does it work?
Users obtain EAS attestations, link them to ProofPay, and can then transact with other verified users.

### Is it secure?
Yes. All payments are validated through smart contracts and both parties must have valid attestations.

## Technical

### What blockchain does ProofPay use?
Base (Ethereum L2) for fast and cheap transactions.

### Can I use my own tokens?
Yes, ProofPay supports both native ETH and ERC20 tokens.

### Are smart contracts audited?
Audits are planned before mainnet launch.

## Usage

### How do I get started?
1. Connect your wallet
2. Obtain an EAS attestation
3. Link attestation to ProofPay
4. Start transacting!

### What wallets are supported?
MetaMask, WalletConnect, and other Web3 wallets.

### Is there a fee?
Only network gas fees. No platform fees currently.
